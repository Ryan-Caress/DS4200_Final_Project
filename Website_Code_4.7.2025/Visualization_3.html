Visualization #3:

Interactive Bar Plot (Shooting % by Position by Team by Year)

This visualization represents average player data by position as a bar plot. Additionally, there are dynamic filters embedded within the visualization that allow you to sort the data. Specifically, you can see the average True Shooting %, Avg 3 Point shooting %, Average 2 point shooting %, and the Average Free Throw Shooting % by position type (Center, Point Guard, Small Forward, PowerForward, and Shooting Guard). Additionally, these stats sorted on a team wide basis, and are looked at for the years 2010 - 2017.
